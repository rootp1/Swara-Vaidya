# Swara-Vaidya
Svara-Vaidya is a web-based tool that analyzes Sanskrit mantra chanting and evaluates pitch and rhythm accuracy. Using audio processing with Librosa and inspired by Vedic Svara, it compares user chants with a reference and generates a Resonance Score.
